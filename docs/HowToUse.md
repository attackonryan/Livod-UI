# Livod-ui

## 介绍

Livod-ui 基于 React-overlays，致力于提供最轻量级的 UI 框架，且提供最大程度与 Ant-design 兼容的 API。

### 导入

方式一： 将源代码拉取至本地，使用 npm i 安装相关依赖，运行 npm run build。根目录下生成 esm 文件，即可使用(需导入 react-overlays 包)。

方式二： npm（后续）

#### DEMO 运行

安装好依赖，使用 npm run dev，自行修改 index.tsx 测试组件

（后续预置对应场景的 DEMO）
